1. Java Lambda
 * hewy!
 ```java
    public int getTotalPrice(){
        int totalPrice=0;
        for (OrderItem orderItem: orderItems){
            totalPrice += orderItem.getTotalPrice();
        }
        return totalPrice;
    }
 ```

  * Using Lambda:
```java
    public int getTotalPrice(){
        int totalPrice= orderItems.stream().mapToInt(OrderItem::getTotalPrice).sum();
        return totalPrice;
    }
```
